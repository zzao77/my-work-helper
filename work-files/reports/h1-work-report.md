# 기획팀 상반기 업무 보고 (초안)

(교육용 가상 샘플 — 실명·실제 수치 아님)
작성: 기획팀 / 기간: 2026년 1~6월
※ 아직 초안. 수치 몇 개 재확인 필요, 팀장님 검토 전임.

## 1. 주요 성과
- 상반기 캠페인 3건 기획·운영 (봄 시즌 2건 + 6월 프로모션 1건)
  - 봄 캠페인은 반응 괜찮았음. 6월 건은 준비기간이 짧아서 좀 아쉬웠음.
- 신규 협력사 2곳 계약 (온누리기획, 대림애드)
- 업무 파일 정리 규칙 도입 → 자료 찾는 시간 줄어든 느낌 (체감상)

## 2. 수치  ※재확인 필요
- 캠페인 참여·도달 수치: 표 따로 (매출표 숫자 대조 중 — 김주임)
- 예산 집행률 약 8x% … 정확한 값 회계팀 확인 후 업데이트할 것

## 3. 진행 중 / 이월 과제
- 하반기 캠페인 기획: 컨셉(가성비 vs 프리미엄) 아직 미정, 협력사와 협의 중
- 하반기 예산안 작성 중 (7월 중순 회계팀 전달 예정)
- 신규 입사자 온보딩 프로그램 정비

## 4. 아쉬웠던 점 / 개선할 것
- 부서 간 일정 공유가 늦어 자료 준비가 촉박했던 적이 몇 번 있었음
- 회의실 예약이 겹치는 문제 (팀 공통 불만)
- 매일 파일이 쌓이는데 정리를 미루면 나중에 찾기 힘듦 → 파일 규칙으로 조금 나아짐

## 5. 하반기 계획 (간단히)
- 캠페인 일정표를 미리 공유해 준비기간 확보
- 협력사 소통 창구 일원화
- (추가할 것 있으면 회의 때 논의)
