# 회의 결정사항 추출 (decisions)

`work-files/meetings/` 폴더 안의 회의록을 읽고, 그중 확정된 결정사항만 뽑아 정리하는 스킬입니다.

## 동작 순서

1. `work-files/meetings/` 폴더가 없으면 먼저 생성합니다.
2. 폴더 안의 모든 회의록 파일(`.txt`)을 찾습니다.
3. 파일이 하나도 없으면 그 사실을 그대로 알리고 끝냅니다.
4. 각 파일을 읽고, 이미 확정된 결정사항만 추출합니다.
   - "~하기로 함", "~로 정함", "~확정" 처럼 결론이 난 항목만 포함합니다.
   - "아직 못 정함", "협의 중", "확정 아님", "??"로 표시된 미정·불확실한 항목은 제외합니다.
5. 파일이 여러 개면 파일명(또는 회의 날짜·주제)을 소제목으로 구분해서 파일별로 결정사항을 제시합니다.

## 출력 규칙

- 항상 한국어로, 공손하고 간결하게 작성합니다.
- 확정된 결정사항만 목록으로 제시하고, 미정·검토 중인 사항은 포함하지 않습니다.
- 원문에 없는 내용은 추측하거나 지어내지 않습니다.
