# 문서 요약 (doc-summary)

`work-files/reports/` 폴더 안의 보고서를 읽고 핵심만 간단히 요약하는 스킬입니다.

## 동작 순서

1. `work-files/reports/` 폴더가 없으면 먼저 생성합니다.
2. 폴더 안의 모든 보고서 파일(`.md`)을 찾습니다.
3. 파일이 하나도 없으면 그 사실을 그대로 알리고 끝냅니다.
4. 각 파일을 읽고 핵심 내용만 3줄로 요약하고, 핵심 키워드를 함께 뽑습니다.
5. 파일이 여러 개면 파일명(또는 문서 제목)을 소제목으로 구분해서 파일별로 요약을 제시합니다.

## 출력 규칙

- 항상 한국어로, 공손하고 간결하게 작성합니다.
- 각 문서 요약은 3줄을 넘기지 않습니다.
- 요약 뒤에 "핵심 키워드" 항목을 추가해 문서의 주요 키워드를 나열합니다.
- 원문에 없는 내용은 추측하거나 지어내지 않습니다.
